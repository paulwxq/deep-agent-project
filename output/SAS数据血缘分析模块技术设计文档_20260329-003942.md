# SAS 数据血缘分析模块技术设计文档

## 1. 项目概述

### 1.1 项目背景

在企业数据仓库和 ETL 流程中，SAS 代码广泛用于数据处理和转换。随着系统演进，需要追踪数据血缘关系以支持：
- 数据质量问题的根因分析
- 变更影响评估
- 合规审计和监管报告

### 1.2 核心目标

设计一个 Python 应用模块，用于分析 SAS 代码和数据集，生成数据血缘分析的对照表，包含：
- **源表名/文件名**：数据来源的标识
- **源表字段**：原始字段名称
- **目标表名/文件名**：数据处理后的输出标识
- **目标表字段**：输出字段名称

### 1.3 设计约束

| 约束项 | 说明 |
|--------|------|
| 技术栈 | Python 3.12 |
| LLM Provider | 可配置（支持 OpenAI、Anthropic、Azure OpenAI、AWS Bedrock 等） |
| 框架选择 | LangChain AI（可选 Deep Agents SDK） |
| 输入规模 | 大量 SAS 代码和文件，不能一次性全部提交给 LLM |
| 输出格式 | CSV 文件 |
| 源数据类型 | Mainframe 定义（主）、CSV/Excel（次） |
| 中间文件 | SAS 文件（.sas7bdat 格式） |

### 1.4 术语定义

| 术语 | 定义 |
|------|------|
| 血缘关系 (Lineage) | 数据字段从源头到目标的转换路径和依赖关系 |
| 源表 (Source Table) | 数据处理的输入表，可以是 CSV、Excel、Mainframe 数据集或 SAS 中间文件 |
| 目标表 (Target Table) | 数据处理的输出表，统一为 CSV 格式 |
| 派生字段 (Derived Field) | 通过计算或转换生成的新字段 |
| 重命名字段 (Renamed Field) | 仅改变名称但保持数据来源不变的字段 |

### 2.2 架构分层说明

#### 2.2.1 应用层
- **配置管理器**：加载和管理 LLM Provider 配置、路径配置、分析参数
- **任务调度器**：协调分析任务执行，支持批量处理和增量分析
- **结果输出管理器**：生成血缘对照表 CSV，支持多种输出格式

#### 2.2.2 分析层
- **SAS 解析器**：静态解析 SAS 代码，提取数据输入输出关系
- **血缘追踪器**：追踪字段级血缘关系，处理重命名和派生逻辑
- **LLM 增强分析器**：对复杂 SAS 代码调用 LLM 进行语义分析

#### 2.2.3 数据层
- **文件扫描器**：扫描和索引 SAS 代码文件及数据文件
- **元数据存储**：存储解析出的表结构、字段定义等元数据
- **血缘关系图存储**：以图结构存储字段级血缘关系

### 2.3 技术选型

| 组件 | 技术选型 | 选型理由 |
|------|----------|----------|
| LLM 框架 | LangChain | 支持多 Provider 配置，有成熟的 Chain 和 Agent 模式 |
| LLM Provider | 可配置（OpenAI/Anthropic/Azure/AWS） | 满足企业不同部署需求 |
| 代码解析 | 正则表达式 + 抽象语法树 (AST) | SAS 无官方 Python AST，采用混合解析策略 |
| 数据存储 | SQLite + NetworkX | 轻量级，支持图结构查询 |
| 配置管理 | Pydantic + YAML | 类型安全，配置验证 |

#### 3.2.3 血缘追踪器模块

```python
# tracker/lineage_graph.py
import networkx as nx
from dataclasses import dataclass
from typing import Dict, List, Set

@dataclass
class LineageNode:
    """血缘图节点"""
    table_name: str
    field_name: str
    node_type: str  # 'source', 'intermediate', 'target'

@dataclass
class LineageEdge:
    """血缘图边"""
    source_node: LineageNode
    target_node: LineageNode
    relationship: str  # 'direct', 'rename', 'derived'
    expression: Optional[str] = None
    sas_file: Optional[str] = None  # 产生此关系的 SAS 文件

class LineageGraph:
    """血缘关系图"""
    
    def __init__(self):
        self.graph = nx.DiGraph()
        self.table_index: Dict[str, Set[str]] = {}  # table -> fields
        
    def add_node(self, node: LineageNode):
        """添加节点"""
        self.graph.add_node(
            (node.table_name, node.field_name),
            node_type=node.node_type
        )
        if node.table_name not in self.table_index:
            self.table_index[node.table_name] = set()
        self.table_index[node.table_name].add(node.field_name)
        
    def add_edge(self, edge: LineageEdge):
        """添加边"""
        self.graph.add_edge(
            (edge.source_node.table_name, edge.source_node.field_name),
            (edge.target_node.table_name, edge.target_node.field_name),
            relationship=edge.relationship,
            expression=edge.expression,
            sas_file=edge.sas_file
        )
    
    def trace_back(self, target_table: str, target_field: str) -> List[LineageNode]:
        """从目标字段回溯到所有源字段"""
        source_nodes = []
        target_node = (target_table, target_field)
        
        # BFS 回溯
        visited = set()
        queue = [target_node]
        
        while queue:
            current = queue.pop(0)
            if current in visited:
                continue
            visited.add(current)
            
            node_data = self.graph.nodes.get(current, {})
            if node_data.get('node_type') == 'source':
                source_nodes.append(LineageNode(
                    table_name=current[0],
                    field_name=current[1],
                    node_type='source'
                ))
            else:
                # 继续回溯前驱节点
                for predecessor in self.graph.predecessors(current):
                    if predecessor not in visited:
                        queue.append(predecessor)
        
        return source_nodes
```

#### 3.2.4 LLM 增强分析器模块

```python
# parser/llm_enhancer.py
from langchain.chat_models import init_chat_model
from typing import List, Optional, Dict, Any
import json

class LLMEnhancementResult:
    """LLM 增强分析结果"""
    field_mappings: List[FieldMapping]
    confidence: float  # LLM 置信度评分
    unresolved_fragments: List[str]  # 无法解析的代码片段
    requires_manual_review: bool  # 是否需要人工复核

class LLMEnhancer:
    """LLM 增强分析器"""
    
    # LLM 调用场景定义
    ENHANCEMENT_SCENARIOS = {
        'complex_expression': '复杂表达式解析（嵌套 IF-THEN-ELSE、宏函数）',
        'macro_expansion': '宏变量展开后的代码片段分析',
        'ambiguous_source': '歧义字段来源确认（MERGE 多表时有相同字段名）',
        'incomplete_parse': '正则解析置信度低于阈值时的补充分析'
    }
    
    def __init__(self, llm_config: LLMConfig):
        self.model = init_chat_model(
            model=llm_config.model_name,
            temperature=llm_config.temperature,
            max_tokens=llm_config.max_tokens
        )
        self.response_schema = self._build_response_schema()
    
    def analyze(self, sas_code: str, context: Dict[str, Any], scenario: str) -> LLMEnhancementResult:
        """
        LLM 增强分析入口方法
        
        参数:
        - sas_code: SAS 代码片段
        - context: 上下文信息（输入表、输出表、已解析映射等）
        - scenario: 调用场景（complex_expression / macro_expansion / ambiguous_source / incomplete_parse）
        
        返回:
        - LLMEnhancementResult: 结构化分析结果
        """
        prompt = self._build_analysis_prompt(sas_code, context, scenario)
        response = self.model.invoke(prompt)
        return self._parse_and_validate_response(response.content, context)
    
    def _build_analysis_prompt(self, sas_code: str, context: Dict[str, Any], scenario: str) -> str:
        """
        构建分析提示词
        
        LLM 输入格式：
        - SAS 代码片段
        - 已解析的输入输出表信息
        - 已解析的字段映射（可能不完整）
        - 待解决的具体问题（根据场景）
        """
        input_tables = context.get('input_tables', [])
        output_tables = context.get('output_tables', [])
        parsed_mappings = context.get('parsed_mappings', [])
        
        return f"""
你是一位 SAS 代码分析专家。请分析以下 SAS 代码，提取字段级血缘关系。

【分析场景】{scenario}

【输入表】
{self._format_tables(input_tables)}

【输出表】
{self._format_tables(output_tables)}

【已解析的字段映射（可能不完整）】
{self._format_mappings(parsed_mappings)}

【待解决的具体问题】
{self._get_scenario_question(scenario)}

【SAS 代码片段】
```sas
{sas_code}
```

【输出要求】
请以 JSON 格式输出，包含以下字段：
- field_mappings: 数组，每个元素包含 source_table, source_field, target_table, target_field, mapping_type, expression
- confidence: 0-1 之间的浮点数，表示分析置信度
- unresolved_fragments: 数组，无法解析的代码片段列表
- requires_manual_review: 布尔值，是否需要人工复核

【输出示例】
{{
    "field_mappings": [
        {{"source_table": "churn_raw", "source_field": "CreditScore", "target_table": "churn_out", "target_field": "credit_scr", "mapping_type": "rename", "expression": ""}},
        {{"source_table": "churn_raw", "source_field": "Balance", "target_table": "churn_out", "target_field": "bal_per_prod", "mapping_type": "derived", "expression": "Balance/num_prod"}}
    ],
    "confidence": 0.85,
    "unresolved_fragments": [],
    "requires_manual_review": false
}}
"""
    
    def _parse_and_validate_response(self, response_content: str, context: Dict[str, Any]) -> LLMEnhancementResult:
        """
        解析 LLM 响应并验证
        
        验证规则：
        1. 检查返回的字段名是否在输入表中存在
        2. 检查输出字段名是否与输出表匹配
        3. 置信度低于 0.5 时标记为需人工复核
        """
        try:
            result_data = json.loads(response_content)
            field_mappings = self._parse_field_mappings(result_data['field_mappings'])
            confidence = result_data.get('confidence', 0.5)
            requires_manual_review = confidence < 0.5 or result_data.get('requires_manual_review', False)
            
            # 验证字段来源
            input_table_fields = {t.name: set(t.fields) for t in context.get('input_tables', [])}
            for mapping in field_mappings:
                if mapping.source_table in input_table_fields:
                    if mapping.source_field not in input_table_fields[mapping.source_table]:
                        # 字段不在源表中，标记为低置信度
                        mapping.confidence = 0.3
                        mapping.ambiguity_flag = True
            
            return LLMEnhancementResult(
                field_mappings=field_mappings,
                confidence=confidence,
                unresolved_fragments=result_data.get('unresolved_fragments', []),
                requires_manual_review=requires_manual_review
            )
        except json.JSONDecodeError:
            # JSON 解析失败，返回空结果
            return LLMEnhancementResult(
                field_mappings=[],
                confidence=0.0,
                unresolved_fragments=[response_content[:200]],
                requires_manual_review=True
            )
    
    def _get_scenario_question(self, scenario: str) -> str:
        """根据场景返回具体问题"""
        questions = {
            'complex_expression': '请解析复杂表达式中每个字段的来源，特别是嵌套的 IF-THEN-ELSE 逻辑。',
            'macro_expansion': '请识别宏变量展开后代码中的表引用和字段映射。',
            'ambiguous_source': 'MERGE 语句涉及多个输入表，请确定每个字段具体来自哪个表。如果无法确定，请标记为 ambiguous。',
            'incomplete_parse': '正则解析未能完整提取字段映射，请补充遗漏的血缘关系。'
        }
        return questions.get(scenario, '请分析字段血缘关系。')
    
    def _format_tables(self, tables: List) -> str:
        """格式化表信息"""
        if not tables:
            return "无"
        return "\n".join([f"- {t.name}: {', '.join(t.fields) if hasattr(t, 'fields') else 'fields unknown'}" for t in tables])
    
    def _format_mappings(self, mappings: List[FieldMapping]) -> str:
        """格式化已解析的映射"""
        if not mappings:
            return "无"
        return "\n".join([
            f"{m.source_table}.{m.source_field} -> {m.target_table}.{m.target_field} ({m.mapping_type})"
            for m in mappings
        ])
    
    def _parse_field_mappings(self, mappings_data: List[Dict]) -> List[FieldMapping]:
        """将 LLM 返回的 JSON 转换为 FieldMapping 对象"""
        return [
            FieldMapping(
                source_field=m['source_field'],
                target_field=m['target_field'],
                source_table=m['source_table'],
                target_table=m['target_table'],
                mapping_type=m['mapping_type'],
                expression=m.get('expression', ''),
                confidence=m.get('confidence', 1.0)
            )
            for m in mappings_data
        ]
    
    def _build_response_schema(self) -> Dict:
        """定义 LLM 响应 JSON Schema"""
        return {
            "type": "object",
            "properties": {
                "field_mappings": {
                    "type": "array",
                    "items": {
                        "type": "object",
                        "properties": {
                            "source_table": {"type": "string"},
                            "source_field": {"type": "string"},
                            "target_table": {"type": "string"},
                            "target_field": {"type": "string"},
                            "mapping_type": {"type": "string", "enum": ["direct", "rename", "derived", "ambiguous"]},
                            "expression": {"type": "string"}
                        },
                        "required": ["source_table", "source_field", "target_table", "target_field", "mapping_type"]
                    }
                },
                "confidence": {"type": "number", "minimum": 0, "maximum": 1},
                "unresolved_fragments": {"type": "array", "items": {"type": "string"}},
                "requires_manual_review": {"type": "boolean"}
            },
            "required": ["field_mappings", "confidence", "requires_manual_review"]
        }
```

**LLM 增强分析边界说明：**

| 项目 | 说明 |
|------|------|
| **输入** | SAS 代码片段 + 上下文（输入表、输出表、已解析映射）+ 场景类型 |
| **输出** | 结构化 JSON：字段映射列表 + 置信度 + 未解析片段 + 人工复核标志 |
| **调用时机** | 正则解析置信度 < 0.7 时；MERGE 语句字段来源歧义时；复杂表达式无法解析时 |
| **降级策略** | JSON 解析失败 → 返回空结果并标记需人工复核；置信度 < 0.5 → 标记需人工复核；字段验证失败 → 降低置信度并标记歧义 |
| **结果合并** | LLM 返回的字段映射与正则解析结果合并；冲突时优先采用 LLM 结果（如果置信度 > 0.7） |

# scanner/file_scanner.py
from pathlib import Path
from typing import List, Dict, Set
from dataclasses import dataclass

@dataclass
class FileInfo:
    path: str
    file_type: str  # 'sas', 'csv', 'excel', 'sas7bdat'
    is_output: bool = False
    is_source: bool = False
    referenced_by: List[str] = field(default_factory=list)

class FileScanner:
    """文件扫描器"""
    
    def __init__(self, root_dir: str):
        self.root_dir = Path(root_dir)
        self.files: Dict[str, FileInfo] = {}
        self.output_csvs: Set[str] = set()
        self.sas_files: Set[str] = set()
        
    def scan(self) -> Dict[str, FileInfo]:
        """扫描目录并构建文件索引"""
        # 第一步：扫描所有 SAS 文件
        for sas_file in self.root_dir.rglob("*.sas"):
            self.sas_files.add(str(sas_file))
            self.files[str(sas_file)] = FileInfo(
                path=str(sas_file),
                file_type='sas'
            )
        
        # 第二步：解析 SAS 文件，识别输入输出
        for sas_file in self.sas_files:
            self._analyze_sas_file_references(sas_file)
        
        # 第三步：标记源文件和输出文件
        self._classify_files()
        
        return self.files
    
    def _analyze_sas_file_references(self, sas_file: str):
        """分析 SAS 文件中的数据引用"""
        with open(sas_file, 'r', encoding='utf-8') as f:
            content = f.read()
        
        # 提取 PROC EXPORT 的输出 CSV
        export_pattern = r'PROC\s+EXPORT[^;]+OUTFILE\s*=\s*["\']?([^"\';\s]+\.csv)'
        for match in re.finditer(export_pattern, content, re.IGNORECASE):
            csv_path = match.group(1)
            self.output_csvs.add(csv_path)
            if csv_path in self.files:
                self.files[csv_path].is_output = True
            else:
                self.files[csv_path] = FileInfo(
                    path=csv_path,
                    file_type='csv',
                    is_output=True
                )
        
        # 提取 PROC IMPORT 的输入文件
        import_pattern = r'PROC\s+IMPORT[^;]+DATAFILE\s*=\s*["\']?([^"\';\s]+)'
        for match in re.finditer(import_pattern, content, re.IGNORECASE):
            input_path = match.group(1)
            if input_path in self.files:
                self.files[input_path].is_source = True
            else:
                self.files[input_path] = FileInfo(
                    path=input_path,
                    file_type=self._detect_file_type(input_path),
                    is_source=True
                )
        
        # 提取 DATA/SET 语句的表引用
        self._extract_data_step_references(content, sas_file)
    
    def _extract_data_step_references(self, content: str, sas_file: str):
        """提取 DATA STEP 中的表引用"""
        # DATA 语句 - 输出表
        data_pattern = r'DATA\s+([a-zA-Z_][a-zA-Z0-9_.]*)'
        for match in re.finditer(data_pattern, content, re.IGNORECASE):
            table_name = match.group(1)
            # 记录临时 SAS 文件
            temp_file = f"{sas_file}:{table_name}"
            self.files[temp_file] = FileInfo(
                path=temp_file,
                file_type='sas7bdat',
                is_temporary=True
            )
        
        # SET/MERGE 语句 - 输入表
        set_pattern = r'(?:SET|MERGE)\s+([a-zA-Z_][a-zA-Z0-9_.\s]+?)(?:\s*\(|\s*;)'
        for match in re.finditer(set_pattern, content, re.IGNORECASE):
            tables = match.group(1).strip().split()
            for table in tables:
                table = table.split('(')[0].strip()  # 去除选项
                # 记录表引用关系
                if table in self.files:
                    self.files[table].referenced_by.append(sas_file)
```

### 4.3 SAS 代码解析算法

```python
# parser/sas_parser.py
import re
from typing import List, Tuple, Optional

class SASParser:
    """SAS 代码解析器"""
    
    # 正则模式定义
    PATTERNS = {
        'data_step': r'DATA\s+([a-zA-Z_][a-zA-Z0-9_.]*)\s*;',
        'set_statement': r'SET\s+([a-zA-Z_][a-zA-Z0-9_.\s]+?)(?:\s*\(|\s*;)',
        'merge_statement': r'MERGE\s+([a-zA-Z_][a-zA-Z0-9_.\s]+?)(?:\s*\(|\s*;)',
        'rename_clause': r'RENAME\s*=\s*\(([^)]+)\)',
        'keep_statement': r'KEEP\s+([a-zA-Z_][a-zA-Z0-9_\s]+?);',
        'assignment': r'([a-zA-Z_][a-zA-Z0-9_.]*)\s*=\s*([^;]+);',
        'proc_export': r'PROC\s+EXPORT[^;]+DATA\s*=\s*([a-zA-Z_][a-zA-Z0-9_.]*)[^;]+OUTFILE\s*=\s*["\']?([^"\';\s]+\.csv)',
        'proc_import': r'PROC\s+IMPORT[^;]+DATAFILE\s*=\s*["\']?([^"\';\s]+)["\']?[^;]+OUT\s*=\s*([a-zA-Z_][a-zA-Z0-9_.]*)',
        # Mainframe 数据集定义解析
        'infile_statement': r'INFILE\s+["\']?([^"\';\s]+)["\']?(?:\s+RECFM\s*=\s*(\w+))?(?:\s+LRECL\s*=\s*(\d+))?',
        'input_statement': r'@(\\d+)\s+([a-zA-Z_][a-zA-Z0-9_]*)\s+([A-Z0-9$.()]+)',
        # SAS 宏变量定义和使用
        'let_statement': r'%LET\s+([a-zA-Z_][a-zA-Z0-9_]*)\s*=\s*([^;]+);',
        'macro_var_ref': r'&([a-zA-Z_][a-zA-Z0-9_]*)(?:\.)?',
    }
    
    def parse_file(self, file_path: str) -> SASProgram:
        """解析单个 SAS 文件"""
        with open(file_path, 'r', encoding='utf-8') as f:
            content = f.read()
        
        program = SASProgram(file_path=file_path)
        
        # 1. 解析 PROC IMPORT（输入）
        self._parse_proc_import(content, program)
        
        # 2. 解析 PROC EXPORT（输出）
        self._parse_proc_export(content, program)
        
        # 3. 解析 DATA STEP
        self._parse_data_steps(content, program)
        
        # 4. 计算解析置信度
        program.parse_confidence = self._calculate_confidence(program)
        
        return program
    
    def _parse_data_steps(self, content: str, program: SASProgram):
        """解析 DATA STEP 语句"""
        # 查找所有 DATA STEP 块
        data_step_pattern = r'DATA\s+([a-zA-Z_][a-zA-Z0-9_.]*)\s*;(.+?)(?=DATA\s+|PROC\s+|RUN\s*;|$)'
        
        for match in re.finditer(data_step_pattern, content, re.IGNORECASE | re.DOTALL):
            output_table = match.group(1)
            step_content = match.group(2)
            
            # 解析输入表
            input_tables = self._extract_input_tables(step_content)
            program.input_tables.extend(input_tables)
            
            # 解析字段映射
            field_mappings = self._extract_field_mappings(
                step_content, 
                input_tables, 
                output_table
            )
            program.field_mappings.extend(field_mappings)
    
    def _extract_field_mappings(
        self, 
        step_content: str, 
        input_tables: List[TableReference],
        output_table: str
    ) -> List[FieldMapping]:
        """提取字段映射关系"""
        mappings = []
        
        # 1. 解析 RENAME 子句
        rename_matches = re.findall(self.PATTERNS['rename_clause'], step_content, re.IGNORECASE)
        for rename_clause in rename_matches:
            # 格式：old1=new1 old2=new2
            pairs = re.findall(r'(\w+)\s*=\s*(\w+)', rename_clause)
            for source_field, target_field in pairs:
                # 假设来自第一个输入表（简化处理）
                source_table = input_tables[0].name if input_tables else "unknown"
                mappings.append(FieldMapping(
                    source_field=source_field,
                    target_field=target_field,
                    source_table=source_table,
                    target_table=output_table,
                    mapping_type='rename'
                ))
        
        # 2. 解析 KEEP 语句
        keep_match = re.search(self.PATTERNS['keep_statement'], step_content, re.IGNORECASE)
        keep_fields = []
        if keep_match:
            keep_fields = keep_match.group(1).split()
        
        # 3. 解析赋值语句（派生字段）
        assignment_matches = re.findall(self.PATTERNS['assignment'], step_content)
        for target_field, expression in assignment_matches:
            if target_field in keep_fields or not keep_fields:
                # 从表达式中提取源字段
                source_fields = self._extract_fields_from_expression(expression)
                for source_field in source_fields:
                    source_table, is_ambiguous = self._find_source_table(source_field, input_tables)
                    mappings.append(FieldMapping(
                        source_field=source_field,
                        target_field=target_field,
                        source_table=source_table or "unknown",
                        target_table=output_table,
                        mapping_type='derived',
                        expression=expression.strip(),
                        ambiguity_flag=is_ambiguous,
                        confidence=0.5 if is_ambiguous else 1.0  # 歧义字段降低置信度
                    ))
        
        # 4. 处理 KEEP 中的直接传递字段
        if keep_fields:
            for keep_field in keep_fields:
                if not any(m.target_field == keep_field for m in mappings):
                    # 直接传递，未重命名
                    source_table, is_ambiguous = self._find_source_table(keep_field, input_tables)
                    mappings.append(FieldMapping(
                        source_field=keep_field,
                        target_field=keep_field,
                        source_table=source_table or "unknown",
                        target_table=output_table,
                        mapping_type='direct',
                        ambiguity_flag=is_ambiguous,
                        confidence=0.5 if is_ambiguous else 1.0
                    ))
        
        return mappings
    
    def _extract_fields_from_expression(self, expression: str) -> List[str]:
        """从表达式中提取字段名"""
        # 简单实现：提取所有标识符
        # 排除数字、函数名、关键字
        keywords = {'in', 'and', 'or', 'not', 'if', 'then', 'else', 'missing'}
        fields = re.findall(r'\b([a-zA-Z_][a-zA-Z0-9_]*)\b', expression)
        return [f for f in fields if f.lower() not in keywords and not f.isdigit()]
    
    def _calculate_confidence(self, program: SASProgram) -> float:
        """计算解析置信度"""
        if not program.field_mappings:
            return 0.0
        
        confidence = 1.0
        
        # 有未解析的输入表
        if not program.input_tables:
            confidence -= 0.3
        
        # 有复杂表达式
        complex_expressions = sum(
            1 for m in program.field_mappings 
            if m.mapping_type == 'derived' and m.expression and len(m.expression) > 50
        )
        if complex_expressions > 0:
            confidence -= 0.1 * min(complex_expressions, 3)
        
        return max(0.0, confidence)
    
    def _parse_mainframe_definition(self, content: str, table_name: str) -> MainframeTableDefinition:
        """
        解析 Mainframe 数据集定义文件（如 def_Churn.sas）
        提取 infile 路径和 input 字段定义
        """
        # 解析 INFILE 语句
        infile_match = re.search(self.PATTERNS['infile_statement'], content, re.IGNORECASE)
        file_path = None
        recfm = None
        lrecl = None
        if infile_match:
            file_path = infile_match.group(1)
            recfm = infile_match.group(2)
            lrecl = int(infile_match.group(3)) if infile_match.group(3) else None
        
        # 解析 INPUT 语句（多行）
        fields = []
        input_pattern = self.PATTERNS['input_statement']
        for match in re.finditer(input_pattern, content, re.IGNORECASE):
            position = int(match.group(1))
            field_name = match.group(2)
            field_format = match.group(3)
            
            # 判断字段类型
            field_type = self._infer_field_type(field_format)
            field_length = self._extract_field_length(field_format)
            
            fields.append(MainframeFieldDefinition(
                name=field_name,
                position=position,
                format=field_format,
                type=field_type,
                length=field_length
            ))
        
        return MainframeTableDefinition(
            table_name=table_name,
            file_path=file_path,
            recfm=recfm,
            lrecl=lrecl,
            fields=fields
        )
    
    def _infer_field_type(self, sas_format: str) -> str:
        """根据 SAS 格式推断字段类型"""
        sas_format_upper = sas_format.upper()
        # 数值型格式
        if any(f in sas_format_upper for f in ['PK', 'FPD', 'COMMA', 'DOLLAR', 'PERCENT', 'BEST']):
            return 'numeric'
        # 字符型格式（包含 $ 符号）
        if '$' in sas_format_upper:
            return 'character'
        # 默认数值型
        return 'numeric'
    
    def _extract_field_length(self, sas_format: str) -> Optional[int]:
        """从 SAS 格式中提取字段长度"""
        # 例如：PK2.0 -> 2, S370FPD6.0 -> 6, $EBCDIC3. -> 3
        match = re.search(r'[\$]?\w*(\d+)', sas_format)
        if match:
            return int(match.group(1))
        return None
    
    def expand_macro_variables(self, content: str, macro_vars: Optional[Dict[str, str]] = None) -> str:
        """
        展开 SAS 宏变量
        支持 %LET 语句解析和 &var 引用展开
        """
        if macro_vars is None:
            macro_vars = {}
        
        # 第一步：扫描所有 %LET 语句，构建宏变量字典
        let_pattern = self.PATTERNS['let_statement']
        for match in re.finditer(let_pattern, content, re.IGNORECASE):
            var_name = match.group(1)
            var_value = match.group(2).strip()
            macro_vars[var_name] = var_value
        
        # 第二步：替换所有 &var 引用
        def replace_macro_var(match):
            var_name = match.group(1)
            return macro_vars.get(var_name, match.group(0))  # 找不到则保留原样
        
        expanded_content = re.sub(
            self.PATTERNS['macro_var_ref'],
            replace_macro_var,
            content
        )
        
        return expanded_content, macro_vars
    
    def _find_source_table(
        self, 
        field_name: str, 
        input_tables: List[TableReference],
        table_fields_index: Optional[Dict[str, List[str]]] = None
    ) -> Tuple[Optional[str], bool]:
        """
        查找字段的来源表
        返回：(表名，是否有歧义)
        
        参数:
        - field_name: 字段名
        - input_tables: 输入表列表
        - table_fields_index: 表名字段列表索引（用于精确匹配）
        """
        if table_fields_index is None:
            # 没有字段索引时，只能基于字段名猜测
            candidate_tables = []
            for table in input_tables:
                candidate_tables.append(table.name)
            
            if len(candidate_tables) == 1:
                return candidate_tables[0], False
            elif len(candidate_tables) > 1:
                # 多表时无法确定，标记歧义
                return candidate_tables[0], True
            else:
                return None, False
        
        # 有字段索引时，精确查找
        candidate_tables = []
        for table_name, fields in table_fields_index.items():
            if field_name in fields:
                candidate_tables.append(table_name)
        
        if len(candidate_tables) == 1:
            return candidate_tables[0], False
        elif len(candidate_tables) > 1:
            # 多表都有该字段，标记歧义
            return candidate_tables[0], True
        else:
            # 字段不在任何输入表中，可能是派生字段或计算变量
            return None, False
```

# 任务调度器实现批量处理
class TaskScheduler:
    """任务调度器"""
    
    def __init__(self, config: AnalysisConfig):
        self.batch_size = config.batch_size
        self.max_llm_calls = config.max_llm_calls_per_file
        self.llm_call_count = 0
    
    def schedule_analysis(self, sas_files: List[str]) -> List[AnalysisTask]:
        """调度分析任务"""
        tasks = []
        
        # 按批次分组
        batches = [
            sas_files[i:i + self.batch_size] 
            for i in range(0, len(sas_files), self.batch_size)
        ]
        
        for batch_idx, batch in enumerate(batches):
            for sas_file in batch:
                task = AnalysisTask(
                    file_path=sas_file,
                    priority=self._calculate_priority(sas_file),
                    requires_llm=False  # 初始不启用 LLM
                )
                tasks.append(task)
        
        return tasks
    
    def should_use_llm(self, parse_result: SASProgram) -> bool:
        """判断是否需要调用 LLM"""
        if self.llm_call_count >= self.max_llm_calls:
            return False
        
        if parse_result.parse_confidence < 0.7:
            self.llm_call_count += 1
            return True
        
        return False

```python
# tracker/lineage_graph.py
class LineageTracker:
    """血缘追踪器"""
    
    def __init__(self):
        self.graph = LineageGraph()
        self.program_results: Dict[str, SASProgram] = {}
    
    def build_graph(self, program_results: Dict[str, SASProgram]):
        """构建血缘关系图"""
        self.program_results = program_results
        
        # 1. 添加所有输入表节点（源节点）
        for program in program_results.values():
            for table in program.input_tables:
                if table.is_source or table.is_temporary:
                    self._add_table_nodes(table, 'source')
        
        # 2. 添加所有输出表节点（目标/中间节点）
        for program in program_results.values():
            for table in program.output_tables:
                node_type = 'intermediate' if table.is_temporary else 'target'
                self._add_table_nodes(table, node_type)
        
        # 3. 添加字段映射边
        for program in program_results.values():
            for mapping in program.field_mappings:
                self._add_mapping_edge(mapping, program.file_path)
    
    def _add_table_nodes(self, table: TableReference, node_type: str):
        """添加表的所有字段节点"""
        fields = self._extract_table_fields(table)
        
        for field in fields:
            node = LineageNode(
                table_name=table.name,
                field_name=field,
                node_type=node_type
            )
            self.graph.add_node(node)
    
    def _extract_table_fields(self, table: TableReference) -> List[str]:
        """
        提取表的字段列表
        根据表类型采用不同策略：
        - mainframe: 从数据集定义文件的 input 语句提取
        - sas7bdat (临时): 从 DATA STEP 的 KEEP 语句或赋值语句推断
        - csv: 读取 CSV 表头
        """
        # 检查缓存
        if table.name in self._field_cache:
            return self._field_cache[table.name]
        
        fields = []
        
        if table.type == 'mainframe':
            # 从 Mainframe 定义文件提取
            fields = self._extract_mainframe_fields(table)
        elif table.type == 'sas7bdat' or table.is_temporary:
            # 临时 SAS 文件：从创建此表的 DATA STEP 推断
            fields = self._extract_temporal_sas_fields(table)
        elif table.type == 'csv':
            # CSV 文件：读取表头
            fields = self._extract_csv_headers(table)
        else:
            # 未知类型，尝试通用提取
            fields = self._extract_fields_generic(table)
        
        # 缓存结果
        self._field_cache[table.name] = fields
        return fields
    
    def _extract_mainframe_fields(self, table: TableReference) -> List[str]:
        """从 Mainframe 数据集定义文件提取字段"""
        # 在 program_results 中查找定义此表的 SASProgram
        for program in self.program_results.values():
            if table.name in program.mainframe_definitions:
                mainframe_def = program.mainframe_definitions[table.name]
                return [f.name for f in mainframe_def.fields]
        
        # 如果未找到定义，返回空列表（将依赖 LLM 增强）
        return []
    
    def _extract_temporal_sas_fields(self, table: TableReference) -> List[str]:
        """从临时 SAS 文件的 DATA STEP 推断字段"""
        fields = set()
        
        # 查找创建此表的 SASProgram
        for program in self.program_results.values():
            for out_table in program.output_tables:
                if out_table.name == table.name:
                    # 从 field_mappings 中提取目标字段
                    for mapping in program.field_mappings:
                        if mapping.target_table == table.name:
                            fields.add(mapping.target_field)
                    
                    # 从 KEEP 语句补充（如果有）
                    # 注：KEEP 语句已在解析阶段提取到 field_mappings
                    break
        
        return list(fields)
    
    def _extract_csv_headers(self, table: TableReference) -> List[str]:
        """读取 CSV 文件的表头作为字段列表"""
        if not table.path:
            return []
        
        try:
            import pandas as pd
            df = pd.read_csv(table.path, nrows=0)
            return list(df.columns)
        except Exception:
            # 文件不存在或无法读取，返回空列表
            return []
    
    def _extract_fields_generic(self, table: TableReference) -> List[str]:
        """通用字段提取方法（降级策略）"""
        # 尝试从所有 field_mappings 中收集此表的字段
        fields = set()
        for program in self.program_results.values():
            for mapping in program.field_mappings:
                if mapping.source_table == table.name:
                    fields.add(mapping.source_field)
                if mapping.target_table == table.name:
                    fields.add(mapping.target_field)
        return list(fields)
    
    # 字段缓存，避免重复提取
    _field_cache: Dict[str, List[str]] = {}
    
    def _add_mapping_edge(self, mapping: FieldMapping, sas_file: str):
        """添加字段映射边"""
        source_node = LineageNode(
            table_name=mapping.source_table,
            field_name=mapping.source_field,
            node_type='source'
        )
        target_node = LineageNode(
            table_name=mapping.target_table,
            field_name=mapping.target_field,
            node_type='target'
        )
        
        edge = LineageEdge(
            source_node=source_node,
            target_node=target_node,
            relationship=mapping.mapping_type,
            expression=mapping.expression,
            sas_file=sas_file
        )
        self.graph.add_edge(edge)
```

# tracker/field_tracer.py
class FieldTracer:
    """字段追踪器"""
    
    def __init__(self, lineage_graph: LineageGraph):
        self.graph = lineage_graph
    
    def trace_to_source(self, target_table: str, target_field: str) -> List[LineagePath]:
        """
        从目标字段追踪到所有源字段
        返回完整路径列表
        """
        paths = []
        source_nodes = self.graph.trace_back(target_table, target_field)
        
        for source_node in source_nodes:
            # 获取路径
            path = self._get_path(
                (source_node.table_name, source_node.field_name),
                (target_table, target_field)
            )
            paths.append(LineagePath(
                source_table=source_node.table_name,
                source_field=source_node.field_name,
                target_table=target_table,
                target_field=target_field,
                intermediate_steps=path
            ))
        
        return paths
    
    def generate_lineage_report(self, output_tables: List[str]) -> List[LineageRecord]:
        """
        为指定的输出表生成血缘对照表
        """
        records = []
        
        for output_table in output_tables:
            if output_table not in self.graph.table_index:
                continue
            
            for target_field in self.graph.table_index[output_table]:
                paths = self.trace_to_source(output_table, target_field)
                
                for path in paths:
                    records.append(LineageRecord(
                        source_table=path.source_table,
                        source_field=path.source_field,
                        target_table=path.target_table,
                        target_field=path.target_field,
                        transformation_type=self._get_transformation_type(path),
                        expression=self._get_expression(path)
                    ))
        
        return records
```

### 5.4 数据集定义文件与处理代码关联机制

```python
# scanner/file_scanner.py - 增强版
class FileScanner:
    """文件扫描器（增强版）"""
    
    def __init__(self, root_dir: str):
        self.root_dir = Path(root_dir)
        self.files: Dict[str, FileInfo] = {}
        self.output_csvs: Set[str] = set()
        self.sas_files: Set[str] = set()
        # 表名索引：表名 -> 定义文件路径
        self.table_definition_index: Dict[str, str] = {}
        # Mainframe 表定义缓存
        self.mainframe_definitions: Dict[str, MainframeTableDefinition] = {}
    
    def scan(self) -> Dict[str, FileInfo]:
        """扫描目录并构建文件索引"""
        # 第一步：扫描所有 SAS 文件
        for sas_file in self.root_dir.rglob("*.sas"):
            self.sas_files.add(str(sas_file))
            self.files[str(sas_file)] = FileInfo(
                path=str(sas_file),
                file_type='sas'
            )
        
        # 第二步：识别数据集定义文件（def_*.sas 模式）
        self._identify_definition_files()
        
        # 第三步：解析 SAS 文件，识别输入输出
        for sas_file in self.sas_files:
            self._analyze_sas_file_references(sas_file)
        
        # 第四步：标记源文件和输出文件
        self._classify_files()
        
        return self.files
    
    def _identify_definition_files(self):
        """识别并解析数据集定义文件"""
        # 扫描 def_*.sas 模式文件
        for def_file in self.root_dir.rglob("def_*.sas"):
            with open(def_file, 'r', encoding='utf-8') as f:
                content = f.read()
            
            # 解析 DATA 语句获取表名
            data_match = re.search(r'DATA\s+([a-zA-Z_][a-zA-Z0-9_.]*)', content, re.IGNORECASE)
            if data_match:
                table_name = data_match.group(1)
                self.table_definition_index[table_name] = str(def_file)
                
                # 解析 Mainframe 定义
                parser = SASParser()
                mainframe_def = parser._parse_mainframe_definition(content, table_name)
                self.mainframe_definitions[table_name] = mainframe_def
    
    def get_table_fields(self, table_name: str) -> List[str]:
        """获取表的字段列表"""
        # 优先从 Mainframe 定义获取
        if table_name in self.mainframe_definitions:
            return [f.name for f in self.mainframe_definitions[table_name].fields]
        
        # 否则返回空列表（将在解析阶段补充）
        return []
```

**关联机制说明：**

| 步骤 | 说明 | 实现位置 |
|------|------|----------|
| 1. 识别定义文件 | 扫描 `def_*.sas` 模式文件，提取表名 | FileScanner._identify_definition_files |
| 2. 建立表名索引 | 记录每个表名对应的定义文件路径 | table_definition_index 字典 |
| 3. 解析字段定义 | 从 INFILE/INPUT 语句提取字段元数据 | SASParser._parse_mainframe_definition |
| 4. 关联处理代码 | 当处理代码引用临时表时，通过索引查找定义文件 | LineageTracker._extract_mainframe_fields |
| 5. 构建完整血缘 | 将源表字段定义与处理代码输出关联 | LineageTracker.build_graph |

```

```python
class IncrementalAnalyzer:
    """增量分析器"""
    
    def __init__(self, cache_dir: str):
        self.cache_dir = Path(cache_dir)
        self.file_hashes: Dict[str, str] = {}
    
    def get_files_to_analyze(self, sas_files: List[str]) -> List[str]:
        """获取需要重新分析的文件"""
        files_to_analyze = []
        
        for sas_file in sas_files:
            current_hash = self._calculate_file_hash(sas_file)
            
            if sas_file not in self.file_hashes:
                # 新文件
                files_to_analyze.append(sas_file)
            elif self.file_hashes[sas_file] != current_hash:
                # 文件已修改
                files_to_analyze.append(sas_file)
            # else: 文件未变化，使用缓存结果
        
        return files_to_analyze
    
    def cache_result(self, file_path: str, result: SASProgram):
        """缓存分析结果"""
        cache_file = self.cache_dir / f"{Path(file_path).stem}.pkl"
        with open(cache_file, 'wb') as f:
            pickle.dump(result, f)
        self.file_hashes[file_path] = self._calculate_file_hash(file_path)
    
    def load_cached_result(self, file_path: str) -> Optional[SASProgram]:
        """加载缓存结果"""
        cache_file = self.cache_dir / f"{Path(file_path).stem}.pkl"
        if cache_file.exists():
            with open(cache_file, 'rb') as f:
                return pickle.load(f)
        return None
```

```python
from concurrent.futures import ThreadPoolExecutor, as_completed

class ParallelAnalyzer:
    """并行分析器"""
    
    def __init__(self, max_workers: int = 4):
        self.max_workers = max_workers
    
    def analyze_batch(self, files: List[str], parser: SASParser) -> Dict[str, SASProgram]:
        """并行分析文件批次"""
        results = {}
        
        with ThreadPoolExecutor(max_workers=self.max_workers) as executor:
            future_to_file = {
                executor.submit(parser.parse_file, f): f 
                for f in files
            }
            
            for future in as_completed(future_to_file):
                file_path = future_to_file[future]
                try:
                    result = future.result()
                    results[file_path] = result
                except Exception as e:
                    logger.error(f"解析失败 {file_path}: {e}")
        
        return results
```

### 6.5 LLM 调用边界与性能指标

**LLM 调用决策树：**

```
正则解析完成
    │
    ▼
parse_confidence >= 0.7？
    │
    ├── 是 → 不使用 LLM，直接输出结果
    │
    └── 否 → 判断调用场景
            │
            ├── complex_expression → 调用 LLM 分析复杂表达式
            ├── ambiguous_source → 调用 LLM 确认字段来源
            ├── macro_expansion → 调用 LLM 分析宏展开后代码
            └── incomplete_parse → 调用 LLM 补充遗漏映射
```

**LLM 输入输出边界：**

| 项目 | 说明 |
|------|------|
| **输入内容** | SAS 代码片段（≤200 行）+ 输入输出表信息 + 已解析映射 + 场景问题 |
| **输出格式** | 结构化 JSON（field_mappings 数组 + confidence + unresolved_fragments + requires_manual_review） |
| **结果合并** | LLM 结果与正则解析结果合并；冲突时以 LLM 结果为准（confidence > 0.7） |
| **降级处理** | LLM 超时/失败 → 使用正则解析结果并标记需人工复核；JSON 解析失败 → 标记需人工复核 |

**性能评估指标：**

| 指标 | 目标值 | 测量方法 |
|------|--------|----------|
| 单文件正则解析时间 | < 100ms | 不含 LLM 调用 |
| 100 文件批量分析时间 | < 5 分钟 | 含 LLM 调用（30% 文件） |
| LLM 调用比例 | ≤ 30% 总文件数 | 仅低置信度文件调用 |
| 单 LLM 调用耗时 | < 30s | 含网络延迟 |
| 人工复核率 | < 10% 输出记录 | ambiguity_flag=true 或 confidence<0.5 |
| 字段解析准确率 | > 90% | 与人工标注对比 |

```python
class LLMRateLimiter:
    """LLM 调用限流器"""
    
    def __init__(self, max_calls_per_file: int = 3, max_total_calls: int = 100):
        self.max_calls_per_file = max_calls_per_file
        self.max_total_calls = max_total_calls
        self.file_call_count: Dict[str, int] = {}
        self.total_calls = 0
    
    def can_call(self, file_path: str) -> bool:
        """判断是否允许调用"""
        if self.total_calls >= self.max_total_calls:
            return False
        if self.file_call_count.get(file_path, 0) >= self.max_calls_per_file:
            return False
        return True
    
    def record_call(self, file_path: str):
        """记录调用"""
        self.file_call_count[file_path] = self.file_call_count.get(file_path, 0) + 1
        self.total_calls += 1
```
